Browserpad v26.6

Browserpad is a notepad in the browser. It can be installed as an application (PWA).

It is a plain text editor built entirely with HTML, CSS and JavaScript without any backend.

Your notes are automatically stored locally in browser storage.

The source code is based on original Browserpad available on GitHub.

Features:

  • Auto-save to localstorage
  • Search in text
  • Statistics
  • Open a file
  • Save to a file
  • Spellcheck
  • TAB / Shift+TAB support
